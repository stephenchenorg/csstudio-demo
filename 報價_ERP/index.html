二、專案概述

本專案旨在開發一套 SaaS（Software as a Service）多租戶架構 的 BigSeller 類型電商 ERP 系統。系統採用多租戶設計，支援多家企業客戶同時使用，每個租戶資料完全隔離。平台營運方可透過訂閱制向客戶收費，建立持續性收入模式。

🏢 SaaS 多租戶架構特點

  • 多租戶資料隔離：每個企業客戶（租戶）的資料完全獨立，互不可見
  • 訂閱制付費模式：支援月付/年付方案，自動扣款與續約提醒
  • 方案與功能權限：不同訂閱方案開放不同功能模組（基礎版/專業版/企業版）
  • 租戶自助註冊：企業客戶可自行註冊、選擇方案、完成付款
  • 平台管理後台：營運方可管理所有租戶、查看收入報表、調整方案
  • 彈性擴展架構：支援水平擴展，隨用戶增長自動調配資源
  • 統一版本更新：所有租戶同步享有功能更新，無需個別部署

專案亮點

  • 支援 8+ 電商平台 API 整合
  • 完整的 進銷存管理 功能
  • 多倉庫、多貨位的 WMS 倉儲系統
  • 即時數據大屏與 BI 報表分析
  • 完善的 權限管理 與操作日誌
  • 支援 多語言、多幣別 跨境電商需求
  • SaaS 訂閱計費系統 與金流整合

00 SaaS 平台核心模組

▼
🏢 多租戶架構 (Multi-Tenancy)

系統核心架構，確保每個企業客戶（租戶）的資料完全隔離，同時共享系統程式碼。

SaaS 實作建議：
• 採用「單一資料庫 + tenant_id 欄位」架構，所有資料表增加 tenant_id 外鍵
• 使用 Laravel 的 Global Scope 自動過濾租戶資料
• 中間件自動識別當前租戶（透過子網域或登入資訊）
• 大型租戶可選配獨立資料庫實現完全隔離
Laravel Tenancy Global Scope Middleware
💳 訂閱計費系統 (Subscription Billing)

管理租戶的訂閱方案、計費週期、自動扣款、發票開立等功能。

SaaS 實作建議：
• 整合綠界/藍新金流實現自動扣款
• 支援信用卡綁定、超商繳費、ATM 轉帳
• 到期前 7 天自動發送續約提醒
• 提供 14 天免費試用期機制
• 支援升級/降級方案，按比例計算費用
ECPay 綠界 Laravel Cashier Webhook
📊 方案與功能權限管理

定義不同訂閱方案可使用的功能模組、API 呼叫次數、儲存空間等限制。

SaaS 實作建議：
• 建立 Plans 資料表定義方案內容
• 使用 Feature Flags 控制功能開關
• 設計用量計數器（訂單數/SKU數/店鋪數）
• 超過限額時優雅降級或提示升級
  • 基礎版：3 店鋪、1000 SKU、基礎報表
  • 專業版：10 店鋪、5000 SKU、進階報表
  • 企業版：無限店鋪、無限 SKU、完整功能
  • 自訂方案：依需求客製功能組合
Feature Flags Rate Limiting
🖥️ 平台營運管理後台 (Super Admin)

平台營運方專用後台，管理所有租戶、查看營收報表、處理客服問題。

SaaS 實作建議：
• 與租戶後台完全分離的獨立系統
• 可切換進入任一租戶後台（模擬登入）協助排錯
• 營收儀表板顯示 MRR、ARR、Churn Rate 等關鍵指標
• 租戶健康度監控（活躍度、用量、異常）
  • 租戶列表與狀態管理
  • 營收報表與財務統計
  • 系統公告發布
  • 客服工單管理
  • 租戶模擬登入
  • 系統監控與告警
獨立 Admin Panel Impersonation

企業客戶可自行完成註冊、選擇方案、付款、開通全流程。

SaaS 實作建議：
• 設計精美的 Landing Page 介紹功能與定價
• 註冊流程：填資料 → 選方案 → 付款 → 自動開通

01 產品管理模組

▼
🔍 產品採集 (API Product Import)

透過電商平台官方 API 抓取商品資訊，安全合規地快速建立商品資料庫（如果串接平台有提供相關 API）。

SaaS 實作建議：
• 整合各電商平台官方 API（淘寶開放平台、1688 API、蝦皮 Open API 等）
• 每位客戶需自行申請並填入各平台的 API Key / App Secret
• 採集任務放入 Queue 非同步執行，避免阻塞
• 按租戶方案限制每日採集次數
• 圖片下載後上傳至租戶專屬 S3 資料夾
⚠️ 重要說明：
• API Key 由客戶自行申請，系統僅提供設定介面
• API 呼叫費用由廠商（客戶）自行吸收，非本報價包含範圍
  • 平台 API 授權設定介面（OAuth 2.0）
  • API Key / Secret 加密儲存
  • 商品資訊批量同步
  • 圖片批量下載與壓縮
  • 支援淘寶、1688、蝦皮等平台 API
  • 採集歷史記錄與錯誤日誌
Platform API OAuth 2.0 Queue Job S3 Storage Encryption
🚚 產品搬家 (Cross-Platform Migration)

將商品從 A 平台一鍵複製到 B 平台，自動處理欄位映射與格式轉換（如果串接平台有提供相關 API）。

SaaS 實作建議：
• 設計統一的商品中間格式 (Canonical Format)
• 各平台 Adapter 負責格式轉換
• 搬家前預覽差異，確認後批量執行
• 記錄搬家日誌，失敗可重試
Adapter Pattern Batch Processing
🖼️ 圖片空間管理

統一管理所有商品圖片，支援分類、裁剪、壓縮、CDN 加速。

SaaS 實作建議：
• 每個租戶獨立的 S3 Prefix（如 tenant-123/images/）
• 按方案限制儲存空間（基礎版 5GB、專業版 50GB）
• 使用 CloudFront/Cloudflare CDN 加速
• 圖片上傳時自動壓縮與生成縮圖
AWS S3 CloudFront CDN Image Optimization
🔗 多平台信息聯動

修改主商品資訊後，自動同步更新到所有已上架的平台（如果串接平台有提供相關 API）。

SaaS 實作建議：
• 設計「主商品」與「平台商品」的關聯結構
• 可設定聯動規則（價格聯動、庫存聯動、標題聯動）
• 同步任務使用 Queue，記錄同步結果
• 同步失敗時通知用戶並可手動重試
Event Driven Queue Webhook

會根據要串接的平台提供商品上下架跟編輯功能。

• 店鋪授權 Token 加密儲存
• 統一的商品列表頁，可篩選平台/店鋪
• 批量操作（批量上架、批量改價、批量下架）
