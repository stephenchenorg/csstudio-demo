針對不同來源，我們各有對應的技術方案

全自動 LINE OA 客戶留言
客戶透過你的 LINE 官方帳號傳訊息時，LINE 平台會即時透過 Webhook 將訊息推送到我們的伺服器。系統自動接收後，交由 AI 進行分類與草擬回覆，最後推播通知給你。全程零人工介入。

全自動 群組訊息（Bot 在群內）
只要將我們的 Bot 加入群組，群組內所有成員的訊息都會透過 Webhook 被接收。系統可辨識發話者身分，自動分類訊息類型，並在需要時草擬回覆建議。適合內部團隊或客戶群組的訊息監控。

半自動 群組訊息（Bot 不在群內）
LINE 不允許 Bot 讀取未加入的群組訊息。我們的解法：你只需截圖聊天畫面並上傳，系統透過 OCR 自動辨識圖片中的文字訊息，還原對話內容後交由 AI 分類與草擬回覆。截圖即分析，不用手動打字。

半自動 個人私訊（1 對 1）
LINE 不開放個人帳號的 API 存取，任何第三方都無法自動讀取私人聊天記錄。我們的解法：提供「AI 回覆助手」Web 工具，你只需將聊天畫面截圖上傳，系統透過 OCR 辨識對話文字，AI 分析上下文後生成回覆草稿，再由你複製回 LINE 送出。隱私完全由你掌控。
